#  ToyStor

##  О проекте

ToyStor — это Java-приложение для управления магазином игрушек.  
Проект реализует базовую бизнес-логику работы с товарами: хранение, добавление, удаление и поиск игрушек.

Цель проекта — продемонстрировать навыки разработки на Java, работы с Maven,\ подключения статического анализа кода через SonarCloud.

---

## ⚙️ Реализованная функциональность

- Добавление новой игрушки
- Удаление игрушки
- Получение списка всех товаров
- Поиск игрушки по параметрам
- Работа с коллекциями
- Модульная структура проекта
- Сборка через Maven
- Статический анализ качества кода через SonarCloud
